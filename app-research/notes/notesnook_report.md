# Notesnook - Detailed Analysis

## Overview
- **Package**: com.streetwriters.notesnook
- **Version**: 3.3.11 (versionCode 15439)
- **Size**: 34-36 MB (varies by architecture)
- **Min SDK**: Android 7.0 (API 24)
- **Target SDK**: Android 14 (API 34)
- **License**: GPL-3.0

## Key Features

### Security & Privacy
- **End-to-end encryption** using XChaCha20-Poly1305 & Argon2
- Zero-knowledge architecture - even Notesnook can't read your notes
- Open source client AND server
- No tracking or analytics
- Self-hostable sync server option

### Cross-Platform Sync
- Android, iOS
- Windows, Mac, Linux
- Web browser
- Seamless, near-instant sync
- Works offline with later sync

### Note Organization
- Multiple notebooks
- Topics within notebooks
- Tags with custom colors
- Pin important notes
- Favorites system
- Powerful search

### Editor Features
- Rich text editor
- Markdown shortcuts
- Todo/checklist items
- Tables
- Images and video embeds
- Code blocks
- Mathematical formulas
- Links with preview
- Bidirectional note linking

### Export & Import
- Export: PDF, HTML, Markdown, Plain Text
- Import from: Evernote, Standard Notes, Google Keep, and more
- Bulk export options

### Additional Tools
- Web Clipper (browser extension)
- Reminders
- Note history
- Trash with recovery

## Pricing Model
- **Free tier**: Basic features, unlimited notes
- **Pro tier**: $2-9/month
  - Unlimited notebooks
  - Note history
  - Rich embeds
  - Vault for double encryption
  - And more

## Strengths
1. **Best-in-class encryption** - Stronger than AES256
2. **Full cross-platform** - True sync across all devices
3. **100% open source** - Including sync server
4. **Feature-rich** - Competes with Evernote/Notion
5. **Beautiful UI** - Clean, modern design
6. **Privacy-first** - No compromises on security
7. **Self-hostable** - Full control option

## Weaknesses
1. **Large app size** - 34+ MB is significant
2. **Requires account** - Even for local use
3. **Freemium limitations** - Some features locked
4. **Higher Android requirement** - Android 7.0+
5. **Less mature than Evernote** - Still evolving
6. **No desktop apps for free tier** - Web only on free

## UX Assessment
- **Ease of use**: 8/10 - Intuitive but feature-rich
- **Speed/performance**: 8/10 - Good but not instant
- **Design quality**: 9/10 - Modern and polished

## Unique Selling Points
- Only fully open-source E2E encrypted notes app with sync
- Evernote alternative with better privacy
- Self-host option for maximum control
- Growing, active community

## Ideal User
- Privacy-conscious users who need sync
- Users migrating from Evernote
- Teams needing secure collaboration
- Developers who want to self-host
- Power users needing advanced features

## Technical Architecture
- Uses XChaCha20-Poly1305 for encryption
- Argon2 for key derivation
- Local-first with cloud sync
- End-to-end encrypted sync server
